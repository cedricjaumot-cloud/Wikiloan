Les leads WikiLoan et DrMortgage sont d'abord enregistrés dans Supabase, puis envoyés automatiquement dans Odoo.

Paramètres API Odoo
Non configuré
Flux de données

Comment les leads arrivent dans Odoo :

1. Visiteur remplit le formulaire sur WikiLoan ou DrMortgage

2. Lead enregistré dans Supabase (wikiloan_leads ou drmortgage_leads)

3. Lead envoyé automatiquement dans Odoo CRM via API REST

4. Tag "WikiLoan" ou "DrMortgage" ajouté automatiquement

Les leads sont conservés dans Supabase même si Odoo est indisponible.
